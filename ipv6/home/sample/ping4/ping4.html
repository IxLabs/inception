NAME

  ping4 - IPv4 ICMP Echo Request/Reply

TARGET

  IPv4 node (Host and Router)

SYNOPSIS

INITIALIZATION

  Please config the target :
     IP Address : 192.168.0.2
     Netmask    : 255.255.255.0

TEST PROCEDURE

  Tester                      Target
    |                           |
    |-------------------------->|
    |   Echo Request            |
    |                           |
    |                           |
    |<--------------------------|
    |   (ARP request)           |
    |                           |
    |                           |
    |-------------------------->|
    |   (ARP reply)             |
    |                           |
    |                           | 
    |<--------------------------|
    |   Echo Reply              |
    |                           |
    |                           |
    v                           v
  1. Send Echo Request
  2. Wait Echo Reply or ARP request
  3. If ARP request received then send ARP reply, and wait Echo Reply again
  4. Receive Echo Reply

JUDGEMENT

  PASS: Echo Reply Received
