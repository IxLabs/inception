NAME

ping.seq - link local echo request & reply test

TARGET

Host and Router.

SYNOPSIS

ping.seq

DESCRIPTION

  1. Send ICMP Echo Request Packet from Tester to Target.
  2. If Tester receive NS packet form Target, reply NA.
  3. Wait ICMP Echo Reply Packet from Target.

JUDGEMET

If the tester receive ``ICMP Echo Reply'' packet from the target, this test is pass.
